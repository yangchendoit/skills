# 五十K记分工具需求文档

## 1. 项目概述

### 1.1 项目目标
开发一款简单便捷的五十K游戏记分工具，帮助玩家快速记录分数、自动计算级数和胜负结果，提升游戏体验。

### 1.2 目标用户
- 五十K爱好者
- 需要快速记录游戏结果的玩家
- 组织比赛的裁判或记录员

### 1.3 核心价值
- 简单易用：一键记录，自动计算
- 快速准确：实时计算级数和结果
- 趣味性强：增加游戏乐趣

## 2. 功能需求

### 2.1 核心记分功能

#### 2.1.1 分数录入
**功能描述**：记录双方队伍的游戏得分

**输入项**：
- 队伍名称（可自定义，默认"我们"/"他们"）
- 基础分值（5、10、K）：手动输入或快速选择
- 炸弹分值：下拉选择炸弹类型
  - 5张炸弹：200分
  - 6张炸弹：500分
  - 7张炸弹：1000分
  - 8张炸弹：2000分
  - 9张炸弹：5000分
  - 10张炸弹：10000分
  - 11张炸弹：20000分
  - 12张炸弹：50000分
- 游次加分：下拉选择
  - 无游次加分
  - 队友1、2游：+200分（对手-200分）
  - 队友1、3游：+100分（对手-100分）
  - 对手1、2游：对手+200分（我方-200分）
  - 对手1、3游：对手+100分（我方-100分）

**快速录入模式**：
- 常用炸弹一键添加（按钮：+5张炸弹、+6张炸弹等）
- 基础分值快速选择（按钮：+50分、+100分、+150分）
- 游次快速切换（单选框）

#### 2.1.2 自动计算
**功能描述**：根据输入自动计算结果

**计算规则**：
1. 计算总分 = 基础分值 + 炸弹分值 + 游次加分
2. 转换级数 = 总分 ÷ 100（四舍五入）
3. 计算级数差 = 我方级数 - 对方级数
4. 判定结果：
   - 级数差 > 0：我方胜 +级数差级
   - 级数差 < 0：对方胜 +级数差级
   - 级数差 = 0：平局，比较原始分数

**结果显示**：
- 双方总分
- 双方级数
- 级数差
- 胜负结果
- 获胜积分（可配置底分和级数积分）

#### 2.1.3 历史记录
**功能描述**：保存游戏历史记录

**记录内容包括**：
- 游戏时间
- 双方得分明细
- 胜负结果
- 总场次统计

**历史记录操作**：
- 查看历史记录
- 删除单条记录
- 清空所有记录
- 导出记录（文本/图片格式）

### 2.2 辅助工具功能

#### 2.2.1 炸弹记录器
**功能描述**：记录已出的炸弹

**功能点**：
- 记录炸弹张数（5-12张）
- 记录炸弹点数
- 显示已出炸弹列表
- 统计剩余炸弹可能

**炸弹计算器**：
- 输入手中炸弹张数
- 计算炸弹分值
- 提示炸弹是否值得出

#### 2.2.2 赖子计数器
**功能描述**：记录已出赖子数量

**功能点**：
- 赖子总数显示（4张）
- 已出赖子数量输入
- 剩余赖子数量显示
- 赖子使用提示

#### 2.2.3 分值牌记录器
**功能描述**：记录已出的5、10、K

**功能点**：
- 显示总分值（200分）
- 记录已出5、10、K数量
- 计算剩余分值牌
- 分值牌分布统计

### 2.3 趣味性功能

#### 2.3.1 成就系统
**功能描述**：记录玩家成就

**成就类型**：
- 新手成就：完成第一局、完成10局、完成50局
- 技术成就：获得11级、获得15级、获得20级、获得25级及以上
- 炸弹成就：首次出5张炸弹、首次出7张炸弹、首次出10张炸弹
- 游次成就：首次实现1、2游
- 胜利成就：连胜3局、连胜5局、连胜10局
- 翻盘成就：落后10级后翻盘获胜
- 完美成就：单局得分2000+分

**成就显示**：
- 徽章图标
- 成就名称
- 解锁时间
- 成就进度条

#### 2.3.2 统计分析
**功能描述**：游戏数据统计

**统计维度**：
- 总局数统计
- 胜率统计
- 平均级数
- 最高级数
- 常用炸弹统计
- 游次成功率
- 胜负趋势图

**可视化展示**：
- 饼图：胜率分布
- 柱状图：级数分布
- 折线图：胜负趋势
- 雷达图：综合能力评估

#### 2.3.3 趣味提示音
**功能描述**：游戏音效增强趣味性

**音效类型**：
- 出炸弹音效（炸弹Emoji💥）
- 游次音效（胜利号角📯）
- 平局音效（鼓点🥁）
- 大胜音效（烟花🎉）
- 连胜音效（掌声👏）

**设置选项**：
- 开关音效
- 调节音量
- 选择音效类型

#### 2.3.4 排行榜
**功能描述**：多玩家排行榜

**排行维度**：
- 总胜局数
- 总胜率
- 最高级数记录
- 连胜记录
- 炸弹大师（出炸弹最多）

**排行榜功能**：
- 本地排行榜（单设备）
- 添加玩家
- 查看排名
- 数据导出

#### 2.3.5 表情包分享
**功能描述**：生成游戏结果分享图

**分享内容包括**：
- 游戏结果
- 双方得分
- 精彩瞬间（大炸弹、完美游次）
- 成就徽章

**分享功能**：
- 生成图片
- 保存到相册
- 分享到社交平台

## 5. 数据结构设计

### 5.1 游戏记录数据结构

```json
{
  "gameId": "string",           // 游戏ID
  "timestamp": "number",        // 时间戳
  "ourTeam": {
    "name": "string",           // 队伍名称
    "baseScore": "number",      // 基础分值
    "bombScore": "number",      // 炸弹分值
    "tourBonus": "number",      // 游次加分
    "totalScore": "number",     // 总分
    "level": "number"           // 级数
  },
  "theirTeam": {
    "name": "string",
    "baseScore": "number",
    "bombScore": "number",
    "tourBonus": "number",
    "totalScore": "number",
    "level": "number"
  },
  "result": {
    "levelDiff": "number",      // 级数差
    "winner": "string",         // 获胜方
    "winScore": "number"        // 获胜积分
  },
  "bombs": [                    // 炸弹记录
    {
      "team": "string",
      "count": "number",
      "card": "string",
      "score": "number"
    }
  ],
  "achievement": ["string"]     // 解锁成就
}
```

### 5.2 玩家统计数据结构

```json
{
  "playerId": "string",
  "name": "string",
  "totalGames": "number",       // 总局数
  "winGames": "number",         // 胜局数
  "winRate": "number",          // 胜率
  "avgLevel": "number",         // 平均级数
  "maxLevel": "number",         // 最高级数
  "totalBombs": "number",       // 总炸弹数
  "maxWinStreak": "number",     // 最大连胜
  "achievements": ["string"],   // 成就列表
  "createTime": "number",
  "updateTime": "number"
}
```

## 7. 用户体验设计

### 7.1 快速流程

```
开局录入
    ↓
快速添加分数（点击按钮）
    ↓
自动计算级数
    ↓
显示结果 + 音效
    ↓
保存记录 + 检查成就
    ↓
分享结果（可选）
    ↓
再来一局
```

### 7.2 微交互设计

- 按钮点击反馈：缩放动画
- 数字增加动画：数字滚动效果
- 结果展示：卡片翻转动画
- 成就解锁：烟花动画 + 音效
- 胜利提示：大号字体 + 震动反馈

### 7.3 防误操作

- 输入二次确认
- 撤销上一步操作
- 清空数据密码保护
- 历史记录删除确认
