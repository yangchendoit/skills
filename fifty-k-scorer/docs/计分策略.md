# 五十K 计分策略

## 计分机制概览

五十K的核心玩法是**抓分制**，通过出牌获得分值，累计率先达到1000分的队伍获胜。

## 分值牌设定

### 基础分值牌

- **每张5**：5分
- **每张10**：10分
- **每张K**：10分

**总分值计算**：
- 两副牌共8张5：8 × 5 = 40分
- 两副牌共8张10：8 × 10 = 80分
- 两副牌共8张K：8 × 10 = 80分
- **五十K总分值**：40 + 80 + 80 = **200分**

### 炸弹分值

**有效炸弹**（5张及以上）可获得分值：

| 炸弹张数 | 分值 | 备注 |
|---------|------|------|
| 4张炸弹 | 0分 | 可带赖子，是合法炸弹但不记分 |
| 5张炸弹 | 200分 | 可使用赖子组成 |
| 6张炸弹 | 500分 | - |
| 7张炸弹 | 1000分 | 可直接获胜 |
| 8张炸弹 | 2000分 | - |
| 9张炸弹 | 5000分 | - |
| 10张炸弹 | 10000分 | - |
| 11张炸弹 | 20000分 | - |
| 12张炸弹 | 50000分 | 理论最大（8张同点数+4张赖子）|

**炸弹组成方式**（以点数3为例，全场共4张赖子）：
- 4张炸弹：3333 / 333+赖子 / 33+赖子×2 / 3+赖子×3
- 5张炸弹：33333 / 3333+赖子 / 333+赖子×2 / 33+赖子×3
- 6张炸弹：333333 / 33333+赖子 / 3333+赖子×2 / 333+赖子×3
- 7张炸弹：3333333 / 333333+赖子 / 33333+赖子×2 / 3333+赖子×3
- 8张炸弹：33333333 / 3333333+赖子 / 333333+赖子×2 / 33333+赖子×3
- 9张炸弹：333333333 / 33333333+赖子 / 3333333+赖子×2 / 333333+赖子×3
- 10张炸弹：3333333333 / 333333333+赖子 / 33333333+赖子×2 / 3333333+赖子×3
- 11张炸弹：33333333333 / 3333333333+赖子 / 333333333+赖子×2 / 33333333+赖子×3
- 12张炸弹：333333333333 / 33333333333+赖子 / 3333333333+赖子×2 / 333333333+赖子×3

**重要规则**：
- 所有炸弹都可以带赖子组成
- **全场共4张赖子**（2张大王 + 2张配牌），炸弹最多带4张赖子
- **每个点数最多8张**（两副牌各4张）
- 炸弹可以被更大的炸弹打死
- **被打死的炸弹不计分**
- 4张炸弹可压其他牌型，但没有分值

## 获胜条件与结算规则

### 结算触发
- **触发条件**：任意一队的累计分数**超过1000分**时，本局结束并结算
- **获胜目标**：争取更高的级数差，获得更大优势

### 级数转换
累计分数按照**每100分一级**转换为级数：

**转换公式**：级数 = 分数 ÷ 100（四舍五入）

**转换示例**：
- 1049分 ÷ 100 = 10.49 → **10级**
- 1050分 ÷ 100 = 10.50 → **11级**
- 1500分 ÷ 100 = 15.00 → **15级**
- 2055分 ÷ 100 = 20.55 → **21级**

### 游戏结果计算
**计算公式**：
- 级数差 = 我方级数 - 对方级数
- 零级（平局）：获胜积分 = 底分
- 非零级：获胜积分 = 底分 + 级数积分 × 级数差

**结果判定**：
- **级数差 > 0**：我方获胜，级数差为胜出级数
- **级数差 < 0**：对方获胜，级数差为对方胜出级数
- **级数差 = 0**：平局，比较原始分数，分数大者胜

**结算示例**：
- 我方1500分（15级）vs 对方800分（8级）→ 级数差 = **+7级**（我方胜7级）
  - 获胜积分 = 底分 + 级数积分 × 7
- 我方1200分（12级）vs 对方1050分（11级）→ 级数差 = **+1级**（我方胜1级）
  - 获胜积分 = 底分 + 级数积分 × 1
- 我方1050分（11级）vs 对方1050分（11级）→ 级数差 = **0级**，比较原始分数
  - 我方1052分 vs 对方1050分 → 我方胜，获得底分
- 我方1100分（11级）vs 对方2050分（21级）→ 级数差 = **-10级**（对方胜10级）
  - 对方获胜积分 = 底分 + 级数积分 × 10

### 积分规则说明
- **底分**：每局基础积分，平局或小胜时获得的基础奖励
- **级数积分**：每级对应的积分数值，决定胜级的奖励大小
- 具体数值由游戏房间或规则设定

### 获胜策略
- **目标**：通过抓分值牌、炸弹分值、游次加分积累高分
- **级数意识**：
  - 接近临界点时需注意（如1049分是10级，1050分是11级）
  - 争取更高的级数，获得更大级数差
- **优势策略**：扩大级数差距，争取更多胜级数

## 游次加分规则

### 游次定义

当有玩家出完手中所有牌时，按顺序记录游次：

- **1游**：第一个出完牌的玩家
- **2游**：第二个出完牌的玩家
- **3游**：第三个出完牌的玩家
- **4游**：第四个出完牌的玩家（最后一个）

### 游次加分表

| 游次组合 | 加分规则 | 战略价值 |
|---------|---------|---------|
| 队友1、2游 | 该队+200分，对手-200分 | 绝对优势，总分差距400分 |
| 队友1、3游 | 该队+100分，对手-100分 | 中等优势，总分差距200分 |
| 其他情况 | 无额外加分 | 仅按实际抓分计算 |

**重要**：游次加分是额外奖励，需加上炸弹分值和分值牌得分。

### 游次示例

**示例一：队友1、2游**
- 队友A第1个出完（1游）
- 队友B第2个出完（2游）
- 结果：该队+200分，对手-200分

**示例二：队友1、3游**
- 队友A第1个出完（1游）
- 对手C第2个出完（2游）
- 队友B第3个出完（3游）
- 结果：该队+100分，对手-100分

**示例三：队友2、4游**
- 对手C第1个出完（1游）
- 队友A第2个出完（2游）
- 对手D第3个出完（3游）
- 队友B第4个出完（4游）
- 结果：对手实现1、3游 → 对手+100分，该队-100分

## 策略建议

### 1. 抓分策略

#### 分值牌抓取
- **优先抓取高分牌**：10和K（各10分）比5（5分）价值更高
- **分值牌总量**：全场基础分值牌共200分
- **每轮评估**：当对手出分值牌时，评估是否值得压牌

#### 炸弹抓分
- **5张炸弹**：200分（相当于所有5、10、K的总分值）
- **6张炸弹**：500分（已过半胜利分数）
- **7张炸弹**：1000分（可直接获胜）
- **策略**：使用炸弹时，确保获得足够分值

#### 分值管理
- 统计当前双方累计分值
- 根据分值差距调整策略
  - 领先时求稳
  - 落后时激进
- **目标**：单局平均需要抓取500分才能获胜

### 2. 出牌策略

#### 首攻策略
- **手牌较好时**：
  - 可出中等牌型试探
  - 保留大牌用于关键时刻

- **手牌较差时**：
  - 出小牌消耗对手
  - 等待队友创造机会

#### 跟牌策略
- 根据对手出牌调整
- 适时选择"不要"而非压牌
- 为队友保留接风机会

### 3. 游次争先策略

#### 游次重要性
- **1、2游**：该队+200分，对手-200分，总分差距400分
- **1、3游**：该队+100分，对手-100分，总分差距200分
- **争先策略**：优先让一个队友出完，争取游次加分

#### 1游策略
- **手牌较好时**：主动争取1游，快速出完
- **手牌较差时**：为队友创造1游机会
- **1游价值**：确保至少1、3游（+100分）

#### 2游配合
- **队友1游后**：全力争取2游，获得+200分奖励
- **对手2游**：阻断对手1、2游，争取1、3游
- **出牌节奏**：根据队友位置调整出牌策略

#### 游次计算
- 实时计算双方可能的游次
- 牺牲个人出完顺序，确保团队游次优势
- 1、2游价值相当于一个5张炸弹（200分）

### 4. 团队配合

#### 接风配合
- 当队友出牌无人压制时，主动选择"不要"
- 让队友获得连续出牌权
- 有利于队友快速出完手牌
- **配合游次**：接风可帮助队友争取1游或2游

#### 牺牲策略
- 当队友手牌较好时，可用大牌为队友开路
- 队友领先时可选择保守策略
- **游次优先后出**：如果队友接近出完，优先确保队友游次

#### 信息传递
- 通过出牌传递信息给队友
- 如：先出特定牌型暗示手牌情况
- **游次暗示**：通过出牌节奏暗示队友是否争取1游

### 4. 赖子牌使用

#### 赖子价值
- **赖子数量**：全场共4张赖子（2张大王 + 2张配牌）
- **赖子作用**：可当做任意牌使用
- **战略价值**：极高

#### 赖子用途选择
1. **组成炸弹**：
   - 5张炸弹可获200分
   - 优先级：组成高分炸弹 > 其他用途

2. **组成五十K**：
   - 如手中有5+10，可用赖子当K
   - 灵活性高，但分值不如炸弹

3. **应急补牌**：
   - 补成对子、三张等牌型
   - 用于压制对手关键牌

#### 赖子注意事项
- 记住已出的赖子数量（最多4张：2张大王 + 2张配牌）
- 推测对手是否有赖子
- 合理分配四张赖子的用途

### 5. 炸弹使用

#### 炸弹分值策略
- **4张炸弹**（0分）：用于控场和压制，优先用小牌组成
- **5张炸弹**（200分）：用于抓取高分值牌或关键时刻
- **6张炸弹**（500分）：可直接扭转局势，价值极高
- **7张炸弹**（1000分）：可直接获胜，必须保留到最后
- **8张及以上炸弹**：超高分值，极其稀有，需谨慎使用

#### 炸弹被打死风险
- **炸弹可被覆盖**：更大的炸弹可以打死当前炸弹
- **被打死不计分**：出炸弹后如果被打死，该炸弹不计分
- **风险评估**：
  - 出炸弹前评估对手是否有更大炸弹
  - 对手手牌少时，炸弹被打死风险高
  - 记牌推测对手剩余炸弹数量和点数

#### 炸弹时机
- **抓分**：对手出大量分值牌时（尤其是10、K）
- **控场**：关键时刻夺取出牌权
- **救场**：队友被压制时解围
- **炸弹对决**：用大炸弹压制对手小炸弹

#### 炸弹保留
- **四张炸弹**：没有分值但可压牌，灵活使用
- **高分炸弹**：优先保留5张及以上有分值的炸弹用于关键时刻
- **记炸弹**：记住已出的炸弹数量和点数，推测对手剩余炸弹
- **防被打死**：
  - 避免在对手可能有更大炸弹时出高分炸弹
  - 对手炸弹出完后再出高分炸弹更安全
  - 被打死的炸弹不计分，损失极大

#### 炸弹构造与赖子使用

**构造策略**：
- **使用赖子**：合理分配4张赖子（2张大王 + 2张配牌）构造高分炸弹
- **所有炸弹可带赖子**：包括4张炸弹
- **收集炸弹**：主动收集同点数的牌
- **策略选择**：使用赖子构造炸弹 vs 用于其他牌型
- **炸弹张数考量**：
  - 实体牌+赖子构造更高张数炸弹
  - 避免炸弹被打死，选择合适时机出牌

**炸弹数量限制**：
- 每个点数：两副牌共8张（如8张3）
- 赖子数量：全场共4张（2张大王 + 2张配牌）
- 最大炸弹：8张实体牌+4张赖子=12张

### 6. 记牌技巧

#### 需要记住的牌
- **赖子**：全场共4张（2张大王 + 2张配牌），记住已出几张
- **炸弹**：已出的炸弹数量和点数
- **分值牌**：所有已出的5、10、K
- **大牌**：2、A、K（K也是分值牌）
- **关键牌型**：如五十K的组成牌

#### 记牌方法
- **赖子优先记**：四张赖子（2张大王 + 2张配牌）至关重要
- **分值牌统计**：记住已出的5、10、K数量
  - 5：共8张
  - 10：共8张
  - K：共8张
- **炸弹统计**：记住已出的炸弹，推测剩余炸弹
- **按轮次记忆**：每轮结束后回顾关键牌

### 7. 分值计算

#### 实时分值计算
- **基础分值**：已抓的5、10、K分数
- **炸弹分值**：已获得的炸弹分数
- **对手分值**：推算对手已抓的分数
- **剩余分值**：场内剩余未抓的分值

#### 分值目标
- **结算触发**：任意一队超过1000分时结算
- **基础分值**：全场200分（5、10、K）
- **炸弹分值**：关键得分来源
- **游次加分**：1、2游+200分，1、3游+100分
- **综合策略**：基础分值+炸弹分值+游次加分 = 总得分
- **级数转换**：总得分÷100（四舍五入）= 级数
- **级数差**：我方级数 - 对方级数 = 胜负结果

### 8. 局势判断

#### 判断手牌实力
- **强牌**：有多个有效炸弹（5张+）、赖子、大牌多
- **中等**：有炸弹、有一定大牌
- **弱牌**：无有效炸弹、大牌少、牌型散

#### 判断局势
- 根据双方累计分值
- 根据剩余手牌数量
- 根据队友和对手的牌型暗示
- **重要**：推算剩余炸弹和赖子（全场共4张赖子：2张大王 + 2张配牌）的分布

#### 快速获胜判断
- **7张炸弹**：1000分，可直接达到结算线
- **累计1000+分**：触发结算，开始计算级数
- **级数临界点**：注意分数的临界值
  - 1049分 = 10级
  - 1050分 = 11级（差1分差1级）
  - 950分 = 10级
  - 949分 = 9级
- **游次优势**：1、2游额外+200分，可提升2级
- **高分炸弹**：8张及以上炸弹可大幅提升级数
- **分值差距**：评估是否需要冒险出大牌以争取更高级数

#### 综合得分计算与积分结算

**步骤一：计算总分**
**总分 = 基础分值（5、10、K）+ 炸弹分值 + 游次加分**

**步骤二：转化为级数**
**级数 = 总分 ÷ 100（四舍五入）**

**步骤三：计算级数差**
**级数差 = 我方级数 - 对方级数**

**步骤四：结算积分**
- **级数差 = 0**（平局）：比较原始分数，分数大者获得底分
- **级数差 ≠ 0**（有胜负）：获胜积分 = 底分 + 级数积分 × 级数差

**示例一：非平局结算**
- 我方：基础分值150分 + 炸弹700分 + 游次200分 = **1050分** → **11级**
- 对方：基础分值600分 + 炸弹200分 = **800分** → **8级**
- 级数差：11 - 8 = **+3级**（我方胜3级）
- 获胜积分：底分 + 级数积分 × 3（如底分10，级数积分5，则得25分）

**示例二：平局结算**
- 我方：基础分值120分 + 炸弹1000分 = **1120分** → **11级**
- 对方：基础分值900分 + 炸弹150分 = **1050分** → **11级**
- 级数差：11 - 11 = **0级**（平局）
- 比较原始分数：1120 > 1050，我方胜
- 获胜积分：**底分**（如底分10，则得10分）

**示例三：平局分数相同**
- 我方：**1050分** → **11级**
- 对方：**1050分** → **11级**
- 级数差：**0级**（平局）
- 比较原始分数：1050 = 1050，分数相同
- 结果：**真正的平局**，可能双方各得底分/2或其他约定规则

## 实战案例分析

### 案例一：级数临界点与积分最大化

**情况**：
- 我方累计1045分，对手900分（对手已达9级）
- 己方手牌有一张5张炸弹可出
- 底分10，级数积分5

**策略分析**：
- **出炸弹**：获得200分 → 总分1245分 → 12级
  - 级数差：12级 - 9级 = **+3级**
  - 获胜积分：10 + 5×3 = **25分**
- **不出炸弹**：总分1045分 → 10级
  - 级数差：10级 - 9级 = **+1级**
  - 获胜积分：10 + 5×1 = **15分**

**结论**：出炸弹可多获得10分积分，应该出炸弹扩大级数优势

### 案例二：稳健抓分与游次配合

**情况**：
- 我方累计850分，对手900分
- 己方队友手牌较好，有机会争取1、2游

**策略**：
- 稳健出牌，避免冒险
- 优先抓取分值牌
- 配合队友争取1、2游（额外+200分）
- 目标：拿到至少200分以上触发结算（1050分+）

### 案例三：炸弹救场与游次争夺

**情况**：
- 对手即将出完（可能1游），手中有大量分值牌
- 对手累计950分，我方累计800分

**策略**：
- 使用炸弹夺取出牌权
- 阻断对手1游，争取为己方争取1游
- 确保抓取当轮分值
- 配合队友争夺游次加分
- 避免对手达到1000+分触发结算，防止对手获得更大级数优势

### 案例四：接风配合与级数最大化

**情况**：
- 队友出连对667788，手牌即将出完
- 己方手中有更大的连对
- 我方累计980分，对手累计700分
- 底分10，级数积分5

**策略分析**：
- **实现1、2游**：获得+200分游次加分
  - 总分：980 + 200 + 抓分 = 1180+分 → 12级
  - 对手：700分 → 7级
  - 级数差：12级 - 7级 = **+5级**
  - 获胜积分：10 + 5×5 = **35分**
- **不实现1、2游**：无游次加分
  - 总分：980 + 抓分 ≈ 1000+分 → 10-11级
  - 对手：700分 → 7级
  - 级数差：约2-3级
  - 获胜积分：约20-25分

**最佳策略**：
- 选择"不要"，让队友接风
- 队友继续出牌，争取1游
- 己方保留实力，争取2游
- 实现1、2游配对，获得+200分奖励
- 最大化级数差和获胜积分

## 常见错误

### 错误一：盲目压牌
- ❌ 不考虑局势，一味压牌
- ✅ 根据局势判断是否需要压牌

### 错误二：炸弹过早暴露
- ❌ 开局就使用炸弹
- ✅ 保留炸弹用于关键时刻

### 错误三：忽视队友
- ❌ 只顾自己出牌，不考虑队友
- ✅ 与队友配合，创造接风机会

### 错误四：不记牌
- ❌ 随意出牌，不考虑已出的牌
- ✅ 记住关键牌，推测对手牌型
