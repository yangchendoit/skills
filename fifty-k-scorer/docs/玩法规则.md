# 五十K 玩法规则

## 游戏基本配置

### 牌数与人数
- **用牌**：两副扑克牌，共108张（104张普通牌 + 2张大王 + 2张配牌）
- **赖子牌**：2张大王 + 2张配牌作为赖子，全场共4张赖子，可当做任意牌使用
- **人数**：4人
- **组队方式**：对面玩家互为朋友组队，形成2 v 2的对抗格局

### 发牌规则
- 系统自动发牌
- **每人27张牌**
- 发牌后不可更换

## 牌型介绍

### 单牌
- 单张任意牌
- 按点数大小比较

### 对子
- 点数相同的两张牌
- 如：33、KK、22
- 点数大的对子更大

### 连对
- 点数连续的两组及以上对子
- 如：3344、667788、JJQQKK
- 必须连续不间断
- 按最大对子点数比较

### 三张
- 点数相同的三张牌
- 如：333、JJJ、AAA

### 连三张
- 点数连续的两组及以上三张
- 如：333444、777888
- 按最大三张点数比较

### 五十K
- 由单张5、10、K组成
- 属于特殊牌型
- 大小：五十K < 炸弹

### 炸弹
- 点数相同的四张及以上牌
- **可使用赖子组成炸弹**（包括4张炸弹）
- 不区分花色
- **全场共4张赖子**（2张大王 + 2张配牌），因此炸弹最多带4张赖子

#### 炸弹组成示例（以点数3为例）

| 炸弹张数 | 组成方式 | 是否有效 | 分值 |
|---------|---------|---------|------|
| 4张炸弹 | 3333 / 333+赖子 / 33+赖子×2 / 3+赖子×3 | ✓ 合法但无分值 | 0分 |
| 5张炸弹 | 33333 / 3333+赖子 / 333+赖子×2 / 33+赖子×3 | ✓ 有分值 | 200分 |
| 6张炸弹 | 333333 / 33333+赖子 / 3333+赖子×2 / 333+赖子×3 | ✓ 有分值 | 500分 |
| 7张炸弹 | 3333333 / 333333+赖子 / 33333+赖子×2 / 3333+赖子×3 | ✓ 有分值 | 1000分 |
| 8张炸弹 | 33333333 / 3333333+赖子 / 333333+赖子×2 / 33333+赖子×3 | ✓ 有分值 | 2000分 |
| 9张炸弹 | 333333333 / 33333333+赖子 / 3333333+赖子×2 / 333333+赖子×3 | ✓ 有分值 | 5000分 |
| 10张炸弹 | 3333333333 / 333333333+赖子 / 33333333+赖子×2 / 3333333+赖子×3 | ✓ 有分值 | 10000分 |
| 11张炸弹 | 33333333333 / 3333333333+赖子 / 333333333+赖子×2 / 33333333+赖子×3 | ✓ 有分值 | 20000分 |
| 12张炸弹 | 333333333333 / 33333333333+赖子 / 3333333333+赖子×2 / 333333333+赖子×3 | ✓ 有分值 | 50000分 |

**炸弹大小比较**：
- 张数越多越大（6张 > 5张 > 4张）
- 同张数按点数比较（如：4444 > 3333）
- 使用赖子组成的炸弹同样有效

**注意**：
- 4张炸弹是合法炸弹，可以压其他牌型
- 4张炸弹没有分值
- 5张及以上炸弹才有分值
- 炸弹可被更大炸弹打死，被打死的炸弹不计分

## 牌型大小比较

### 点数大小（从大到小）
```
2 > A > K > Q > J > 10 > 9 > 8 > 7 > 6 > 5 > 4 > 3
```

### 牌型大小（从大到小）
```
炸弹 > 五十K > 连三张/三张 > 连对/对子 > 单牌
```

**注意**：不同牌型之间不能互相压（炸弹除外）。

## 出牌规则

### 基本流程

1. **首个出牌**
   - 系统随机确定首个出牌者
   - 首个出牌者可出任意合法牌型

2. **依次出牌**
   - 逆时针方向依次出牌
   - 后出者可选择压牌或不要
   - 压牌必须出同类型的更大的牌

3. **接风规则**
   - 当某玩家出牌后，其他玩家都不要
   - 由该玩家的队友（对面玩家）获得下一轮出牌权
   - 称为"接风"

4. **单局结束**
   - 当一队玩家全部出完手中的牌时，单局结束
   - 开始计分并进入下一局

### 赖子牌规则

- **赖子定义**：两张配牌作为赖子，可当做任意牌使用
- **赖子用途**：
  - 组成对子、三张、连对等牌型
  - 组成炸弹（如：333+赖子=4张炸弹）
  - 组成五十K（如：5+10+赖子=五十K）

### 压牌规则

- **同类型压牌**：必须出相同牌型的更大的牌
  - 单牌压单牌
  - 对子压对子
  - 连对压连对
  - 三张压三张

- **炸弹压牌**：
  - 所有炸弹（包括4张）都可以压其他牌型
  - 小炸弹不能压大炸弹
  - 张数相同时按点数比较
  - 使用赖子组成的炸弹同样有效
  - **炸弹可被覆盖**：更大的炸弹可以打死较小的炸弹
  - **被打死的炸弹不计分**

### 出牌技巧

- **记牌**：记住已出的关键牌（如2、A、K等）
- **配合**：与队友配合，适时接风或压制对手
- **保留实力**：保留炸弹等大牌用于关键时刻
- **控制局势**：通过出牌策略控制主动权

## 游次规则

### 游次定义

当某一队玩家全部出完手中的牌时，单局结束。按照出完牌的顺序记录游次：

- **1游**：第一个出完手中所有牌的玩家
- **2游**：第二个出完手中所有牌的玩家
- **3游**：第三个出完手中所有牌的玩家
- **4游**：第四个出完手中所有牌的玩家（最后一个）

### 游次加分规则

游次会对本局得分产生额外影响：

| 游次组合 | 加分规则 | 说明 |
|---------|---------|------|
| 队友1、2游 | 该队+200分，对手-200分 | 队友连续出完，占据绝对优势 |
| 队友1、3游 | 该队+100分，对手-100分 | 队友分隔出完，仍有优势 |
| 其他情况 | 无额外加分 | 游次无优势，仅按实际抓分计算 |

**示例**：
- 队友A是1游，队友B是2游 → 该队额外+200分，对手额外-200分
- 队友A是1游，队友B是3游 → 该队额外+100分，对手额外-100分
- 队友A是2游，队友B是4游 → 无额外加分

**注意**：游次加分是额外奖励，还需加上正常抓取的炸弹分值和分值牌得分。

## 胜负判定

### 结算触发条件

当任意一队的累计分数**超过1000分**后，本局结束，开始结算分数。

### 级数转换规则

累计分数按照**每100分一级**转换为级数：

**转换公式**：级数 = 分数 ÷ 100（四舍五入）

**转换示例**：
- **1049分** ÷ 100 = 10.49 → 四舍五入 = **10级**
- **1050分** ÷ 100 = 10.50 → 四舍五入 = **11级**
- **1500分** ÷ 100 = 15.00 → **15级**
- **2055分** ÷ 100 = 20.55 → 四舍五入 = **21级**

### 游戏结果计算

用两队的级数相减，得到本轮游戏的胜负结果：

**计算公式**：级数差 = 我方级数 - 对方级数

**结果判定**：
- **级数差 > 0**：我方获胜，级数差为胜出的级数
- **级数差 < 0**：对方获胜，级数差为对方胜出的级数
- **级数差 = 0**：平局，比较原始分数

### 积分结算规则

#### 零级（平局）结算
当级数差为0时：
- 比较双方**原始分数**
- **分数大的一方获胜**
- 获胜方获得**底分**（基础积分）

#### 非零级结算
当级数差不为0时：
- **获胜方积分** = 底分 + 级数积分 × 级数差
- **底分**：每局的基础积分（固定值，如10分）
- **级数积分**：每级对应的积分数值（如5分/级）

**结算示例**：

| 我方分数 | 我方级数 | 对方分数 | 对方级数 | 级数差 | 结果 | 获胜积分（假设底分10，级数积分5）|
|---------|---------|---------|---------|--------|------|--------|
| 1500分 | 15级 | 800分 | 8级 | +7级 | 我方胜7级 | 10 + 5×7 = 45分 |
| 1200分 | 12级 | 1050分 | 11级 | +1级 | 我方胜1级 | 10 + 5×1 = 15分 |
| 1050分 | 11级 | 1050分 | 11级 | 0级 | 平局，比分数 | 获胜方得底分10分 |
| 1052分 | 11级 | 1050分 | 11级 | 0级 | 我方分数高 | 我方胜，得底分10分 |
| 1100分 | 11级 | 2050分 | 21级 | -10级 | 对方胜10级 | 对方得10 + 5×10 = 60分 |

**注意**：底分和级数积分的具体数值由游戏房间或规则设定，上表仅为示例。

### 分值牌设定

#### 基础分值牌
- **每张5**：5分
- **每张10**：10分
- **每张K**：10分
- **总分值**：所有5、10、K共计**200分**

#### 炸弹分值
- **5张炸弹**（可带赖子）：200分
- **6张炸弹**：500分
- **7张炸弹**：1000分
- 更多张数依次倍增

### 获胜条件

**累计分值先达到1000分的一队获得胜利**。

## 常见术语

- **赖子**：2张大王 + 2张配牌，全场共4张，可当做任意牌使用
- **炸弹**：点数相同的四张及以上牌，可带赖子组成
- **有效炸弹**：5张及以上有分值的炸弹
- **接风**：队友出牌后其他人都不要，队友获得出牌权
- **压牌**：出更大的牌压制前家的牌
- **不要**：选择不出牌
- **跑牌**：将手中的牌全部出完
- **1游/2游/3游/4游**：按出完牌顺序排名的游次

## 游戏策略建议

1. **团队配合**
   - 与队友保持默契
   - 为队友创造接风机会
   - 关键时刻用炸弹救场

2. **记牌能力**
   - 记住已出的2、A、K等大牌
   - 推测对手手中可能的大牌
   - 判断炸弹的可能数量

3. **出牌节奏**
   - 控制出牌速度，不宜过快
   - 观察对手出牌习惯
   - 适时改变策略
